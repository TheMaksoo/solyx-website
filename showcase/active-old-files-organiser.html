VS Code extension flow (what you will see)
  1. 1. Scan workspace: Extension reads file last-modified times and groups results by age.
  2. 2. Sidebar view: You get buckets like 30d+, 90d+, 180d+, 365d+ with file counts.
  3. 3. Open stale file: Click an entry to inspect the file before cleanup.
  4. 4. Cleanup pass: Archive, refactor, or keep files intentionally with better context.
